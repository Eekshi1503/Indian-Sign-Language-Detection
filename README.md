# Indian Sign Language Detection System

## Overview
This project is a real-time Indian Sign Language Detection system that recognizes hand gestures using Computer Vision and Deep Learning techniques. It helps in bridging the communication gap between hearing-impaired individuals and others.

## Features
- Real-time hand gesture detection
- Recognizes multiple sign language gestures
- Uses deep learning for classification
- Works with webcam input
- Improves accessibility and communication

## Tech Stack
- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## ⚙️ How It Works
1. Capture hand gesture using webcam
2. Preprocess the image
3. Feed into trained deep learning model
4. Predict the gesture
5. Display the corresponding output
